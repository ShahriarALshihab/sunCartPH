## 📌 Overview

SunCart is a modern summer-themed eCommerce web app where users can browse and explore seasonal products like sunglasses, clothing, skincare, and beach accessories. Users can view product details and access protected pages after logging in.

---

## ✨ Features

* 🔐 Authentication (Email/Password + Google)
* 🛍️ Product listing with 12+ items
* 🔒 Protected product details page
* 👤 User profile page with update option
* 🎯 Hero section with offers & featured products
* 📱 Fully responsive design
